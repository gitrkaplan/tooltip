# Tooltip
A simple tooltip for users who want to display a tooltip based on positioning of button within the viewport.

## Usage

```$ mongod
$ npm install
$ npm run seed
$ npm run start```

## Uses

- HTML5
- CSS3
- Javascript(ES2015+)
- Node.js
- Express.js
- MongoDB
